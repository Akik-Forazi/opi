OPI Documentation

Complete reference for omnip and the OPI REST API.

install

Install a package from OPI or locally.

omnip install math_extra      # from OPI registry
omnip install .               # local directory
omnip install math_extra@1.0  # specific version

Packages install to %USERPROFILE%\.omnikarai\modules\<name>\. Use with use math_extra in your .ok files.

uninstall

omnip uninstall math_extra

list

omnip list

info

omnip info math_extra

search

omnip search math
omnip search "string manipulation"

publish

Requires an API token set as OPI_TOKEN.

set OPI_TOKEN=opi_your_token_here
omnip publish .

update

omnip update              # update all packages
omnip update math_extra   # update one

init

Creates omnikarai.toml in the current directory.

omnip init

omnikarai.toml

[metadata]
name        = "my_package"
version     = "1.0.0"
description = "Does something useful"
author      = "Your Name"
license     = "MIT"
homepage    = "https://github.com/you/my_package"
keywords    = ["util", "math"]

[dependencies]
math_extra = ">=1.0"

API Tokens

Generate tokens in Dashboard → API Tokens. Pass as env var or header:

set OPI_TOKEN=opi_your_token_here   # Windows
export OPI_TOKEN=opi_your_token     # Linux/Mac

# Header usage:
Authorization: Bearer opi_your_token_here

REST API

Base URL: https://opi-nine.vercel.app

MethodEndpointDescription
GET/api/packagesList / search packages
GET/api/packages?q=mathSearch by keyword
GET/api/packages/:namePackage info + versions
POST/api/packagesPublish package (auth)
DELETE/api/packages/:nameDelete (owner only)
GET/api/statsRegistry stats
POST/api/auth/registerCreate account
POST/api/auth/loginSign in → JWT
GET/api/auth/meCurrent user
GET/POST/DELETE/api/user/tokensManage API tokens
PATCH/api/user/profileUpdate profile
GET/api/user/:usernamePublic profile

Example: publish via curl

curl -X POST https://opi-nine.vercel.app/api/packages \
  -H "Authorization: Bearer opi_your_token" \
  -H "Content-Type: application/json" \
  -d '{"name":"my_pkg","version":"1.0.0","description":"Cool pkg","license":"MIT","keywords":["util"],"dependencies":{}}'
