OPI Documentation

Complete reference for omnip and the OPI REST API.

install

Install a package from OPI or locally.

omnip install math_extra      # from OPI registry
omnip install .               # local directory
omnip install math_extra@1.0  # specific version

Packages install to %USERPROFILE%\.omnikarai\modules\<name>\.

uninstall

omnip uninstall math_extra

list

omnip list

info

omnip info math_extra

search

omnip search math
omnip search "string manipulation"

publish

Requires an API token set as OPI_TOKEN.

set OPI_TOKEN=opi_your_token_here
omnip publish .

update

omnip update              # update all packages
omnip update math_extra   # update one

init

Creates omnikarai.toml in the current directory.

omnip init

omnikarai.toml

[metadata]
name        = "my_package"
version     = "1.0.0"
description = "Does something useful"
author      = "Your Name"
license     = "MIT"
homepage    = "https://github.com/you/my_package"
keywords    = ["util", "math"]

[dependencies]
math_extra = ">=1.0"

API Tokens

Generate tokens in Dashboard → API Tokens. Pass as env var or header:

set OPI_TOKEN=opi_your_token_here   # Windows
export OPI_TOKEN=opi_your_token     # Linux/Mac

# Header:
Authorization: Bearer opi_your_token_here
